Patent Number: US623456A2

Patent Title: System and method for security processing media streams

Claim No Key Features of US623456A2 US7620179B2 US9225761B2
1.0 A keystream generator, comprising a feedback shift register structure, having input, intermediate and output stages through which data bits are shifted serially in response to a clock signal,
1.1 and means for varying the polynomial code by applying to the logic elements an enabling polynomial code signal that varies in accordance with the content of data bits shifted from a predetermined register stage
1.2 wherein the logic elements process a data bit shifted from the preceding stage with the data bit fedback from the output stage in accordance with the polynomial code for input into the succeeding register stage
2 means for processing the data bits shifted from a predetermined stage to provide a keystream
3 A keystream generator according to claim 1, wherein the bits that cause the polynomial code signal to be varied are not included in the keystream.
4.0 A keystream generator, comprising a feedback shift register structure, having input, intermediate and output stages through which data bits are shifted serially in response to a clock signal,
4.1 and means for varying the polynomial code by applying to the logic elements an enabling polynomial code signal that varies in accordance with the content of data bits shifted from a predetermined register stage
4.2 wherein the logic elements process a data bit shifted from the preceding stage with the data bit fedback from the output stage in accordance with the polynomial code for input into the succeeding register stage
5 means for processing the data bits shifted from a predetermined stage to provide a keystream
6 A keystream generator according to claim 1, wherein the bits that cause the polynomial code signal to be varied are not included in the keystream.
